## Local development setup

1. Clone this repo
    ```shell
    git clone ___
    ```

1. Create a `.env` file from the example file, and update it with your config
   values.
    ```shell
    cd noteworthy
    cp .env.example .env
    vi .env # or open with your favorite editor, and replace placeholders
    ```
